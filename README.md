# Sistema de Conta Bancária

## Descrição do Projeto
Este projeto consiste no desenvolvimento de um sistema simples de conta bancária, com o objetivo de servir como base para estudos na disciplina de Engenharia de Software. A proposta é evoluir o sistema ao longo do curso, incorporando boas práticas de desenvolvimento.

## Objetivo
Desenvolver um sistema básico que permita compreender conceitos fundamentais de organização de código, testes e evolução incremental de software.

## Tecnologias Utilizadas
- Java
- Maven
- JUnit
